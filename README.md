## 技术
@react16.7.0+@webpack4.28.4+@axios+@scss4.11.0+express
## 项目运行

     git clone https://github.com/huangxiao1009/sing-react.git
     cd sing-react
     npm install
     npm run dev
     ###新开一个git 命令窗口
     cd sing-react
     npm install 
     node server/app.js
